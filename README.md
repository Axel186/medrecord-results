## Before you start
Before you start let's prepare our project.

1. Set configuration file:
```
cp config.default.js config.js
```

1. Install modules by using bower:
```
npm run bower
```

1. Build `dist` folder:
```
npm run build
```

1. Run the server to see the results:
```
npm run server
```

1. Enter to http://localhost:3000/
